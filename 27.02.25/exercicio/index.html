Bem vindos à aula de Desenvolvimento Web!

Hoje vamos aprender os fundamentos do HTML.

O HTML (HyperText Markup Language) é a linguagem base para a construção de paginas web. Ele permite estruturar o conteúdo utilizando diferentes elementos.

Aprender HTML é o primeiro passo para se ronar um desenvolvedor wev. com dedicação e prática você poderá criar sites incríveis!
